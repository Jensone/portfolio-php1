# Portfolio avec PHP

Création d'un portfolio en PHP avec les technologies suivantes :

- HTML
- Tailwind CSS pour le style
- Routeur PHP
- Mailtrap pour envoyer des mails

Liste pages :

- accueil
- presentation
- competences
- projets
- contact
